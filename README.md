## Hunter x Hunter Character Index
A sleek, data-driven character database for the Hunter x Hunter universe. This application fetches real-time data from a dedicated API to provide fans with an interactive way to explore their favorite hunters and villains.

## Core Features
- Character Discovery: Browse through a comprehensive collection of HxH characters presented as interactive cards.

- Real-time Search: Instantly filter characters by name using a dynamic search bar for quick navigation.

- Dynamic Routing: Dedicated detail pages for every character, providing in-depth information such as Nen types, abilities, and lore.

- Type Safety: Built with TypeScript to ensure robust code quality and a predictable developer experience.

## Technical Stack
- Frontend: React & Next.js

- Language: TypeScript

- Styling: TailwindCSS (for a modern, utility-first responsive design)

- Data Fetching: REST API integration

## How to Run
- Install dependencies:

Bash
npm install
- Start the development server:

Bash
npm run dev
- Open [http://localhost:3000](http://localhost:3000) to view the application or Vercel Deploy [https://huntersapp-zeta.vercel.app/]
